üìä Project Summary

For the de-novo analysis of chicken ISREs, 193 ISGs, previously found by Santhakumar et al. in 2019, were chosen. Upstream regions of 2000bp were used as promoters to possibly contain ISREs. That is a large enough window and can be lowered for further optimization. HOMER (Hypergeometric Optimization of Motif EnRichment) was chosen as a program to search the aforementioned upstream regions for new motifs. List of non-ISGs were used as a negative background against the genes ISGs analyzed for the HOMER run. The output motifs were then mapped to the fasta file using HOMER's mapMotifs.pl, which was then sorted by score. The HOMER score is the log-likelyhood ratio (LLR) x 1000
